# Education Center Management System - Presentation

This folder contains a professional PowerPoint presentation that comprehensively analyzes the Education Center Management System project.

## Files

- `generate_pptx.py` - Python script to generate the PowerPoint presentation
- `requirements.txt` - Python dependencies
- `Education_Center_Presentation.pptx` - Generated presentation (after running the script)

## How to Generate the Presentation

1. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the script:
   ```bash
   python generate_pptx.py
   ```

3. The presentation will be saved as `Education_Center_Presentation.pptx`

## Presentation Content

The generated presentation includes:

1. **Title Slide** - Project name and overview
2. **Project Overview** - General description and scope
3. **Problem Statement** - Issues the system solves
4. **Core Features** - Main functionality grid
5. **User Roles** - Student and Teacher capabilities
6. **Payment System** - Paymob integration details
7. **Content Management** - Video player and content types
8. **Assessment System** - Quizzes and assignments
9. **Architecture** - System components diagram
10. **Technology Stack** - Backend, Frontend, Database, Deployment
11. **Database Schema** - Key models and relationships
12. **Key Achievements** - Major accomplishments
13. **Deployment & DevOps** - Docker and infrastructure
14. **Conclusion** - Summary and final thoughts
