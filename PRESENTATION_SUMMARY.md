## Project Overview

**Education Center Management System** is a comprehensive learning platform that connects students and teachers, providing complete management of the educational process from enrollment to graduation.

### Key Findings:
- **Full-Stack Application**: NestJS backend + React frontend
- **Database**: MySQL with Prisma ORM
- **Deployment**: Docker containerization
- **Payment Integration**: Paymob gateway
- **Multi-Role System**: Students, Teachers, Admins
- **Rich Features**: Courses, Quizzes, Assignments, Attendance, Scheduling
